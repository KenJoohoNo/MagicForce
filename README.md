# 강화(MagicForce) 플러그인

- Minecraft Version : 1.21.4

## 명령어
- /강화 : 강화 GUI 를 엽니다.
- /강화권등록 : 일반 강화권 아이템을 등록합니다. `(magicforce.ticket)`
- /확정강화권등록 : 확정 강화권 아이템을 등록합니다. `(magicforce.ticket)`
- /최종강화권등록 : 9강 -> 10강 으로 강화할떄 쓰는 아이템을 등록합니다. `(magicforce.ticket)`
